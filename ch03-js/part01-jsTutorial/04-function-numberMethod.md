### Function

Hàm trong js được chia làm 2 loại chính và chia theo chức năng của nó
Function declaration | Function Expression
Nhưng nếu phân loại một cách đầy đủ theo cú pháp và tính chất hoạt động, chúng ta có 4 loại hàm chính:

- Function Declaration
- Function Expression
- Arrow Function
- Anonymous Function
