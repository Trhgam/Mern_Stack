# MERN Stack Learning Journey

Dự án lưu trữ lộ trình học tập và thực hành Fullstack JavaScript (MERN Stack), bao gồm từ kiến thức cơ bản đến nâng cao.

## 📁 Cấu trúc thư mục

| Thư mục | Nội dung |
| :--- | :--- |
| `ch01-baiTap` | Tổng hợp các bài tập rèn luyện tư duy |
| `ch01-html` | Kiến thức nền tảng về HTML5 và ngữ nghĩa (Semantic HTML) |
| `ch02-css` | Giao diện người dùng, Flexbox, Grid và Responsive |
| `ch03-js` | JavaScript ES6+, DOM, Asynchronous (Promise, Async/Await) |
| `ch04-git` | Quản lý mã nguồn với Git và quy trình làm việc GitHub |
| `ch05-nodejs` | Runtime môi trường server-side và Package Manager (NPM/Yarn) |

## 🛠 Công nghệ sử dụng (MERN)

* **MongoDB:** Cơ sở dữ liệu NoSQL linh hoạt.
* **Express.js:** Web Framework tối ưu cho Node.js.
* **React.js:** Thư viện xây dựng giao diện người dùng (UI).
* **Node.js:** Môi trường thực thi JavaScript phía Server.

## 🚀 Cách bắt đầu

1. **Clone repository:**
   ```bash
   git clone [https://github.com/Trhgam/Mern_Stack.git](https://github.com/Trhgam/Mern_Stack.git)
  ```
2. **Truy cập từng chương:**
Mỗi chương sẽ có các bài tập hoặc project nhỏ riêng biệt. Hãy di chuyển vào từng thư mục để xem chi tiết.

3. **Cài đặt dependencies (nếu có):**
  ```bash
  npm install
  ```
